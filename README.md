# Metagenomic biosurveillance

Metagenomic biosurveillance of SecureBio Zephyr pooled nasal-swab viral reads
(ONT long reads): taxonomy + coverage, and k-mer clustering vs taxonomy.

- **Code and analysis:** [`zephyr-biosurveillance/`](zephyr-biosurveillance/README.md)
  — see that README for the environment, run order, and outputs.
- **Write-up:** the memo (Parts 1–3) is submitted as a Google Doc and mirrored in
  [`zephyr-biosurveillance/memo/memo-core.md`](zephyr-biosurveillance/memo/memo-core.md); a fuller
  version with all figures and tables renders from
  [`zephyr-biosurveillance/memo/memo-extended.Rmd`](zephyr-biosurveillance/memo/memo-extended.Rmd).

> Note: results in this repo reflect a partial pipeline run; the memo and tables are
> consistent with that run.
